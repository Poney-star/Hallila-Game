Adam - Ping Pong Range

Version galerie de tir / clicker :
- viser à la souris
- cliquer pour envoyer une balle de ping-pong vers le point visé
- les cibles se déplacent sur plusieurs lignes comme dans un stand de tir
- un tir raté coûte une vie
- finir tous les niveaux débloque la raquette dorée

Fichiers :
- index.html
- style.css
- js/config.js
- js/audio.js
- js/entities.js
- js/game.js
- assets/target.png (image cible intégrée depuis la photo fournie)

Ouverture :
- ouvrir index.html dans un navigateur récent
